# Nexus Business PR SaaS V2 Pro

Versión más completa y operacional.

## Incluye
- Auth Email/Password Firebase.
- Datos privados por usuario: `users/{uid}/workspace/main`.
- Selector de industria.
- Clientes, servicios, equipo, facturas, cobros, caja, reportes, planes y configuración.
- Facturas PDF con logo y datos corporativos.
- Reportes PDF ejecutivos.
- Flujo servicio -> factura -> cobro -> caja.
- Comisiones por ID único de servicio.
- Botón de mapa por origen/destino.
- Firestore rules por UID.

## Firebase
Activar Authentication > Email/Password.
Copiar `firestore.rules` en Firestore Rules.

## GitHub Pages
Subir `index.html`, `styles.css`, `app.js`, `firestore.rules` y `README.md` al repo.
