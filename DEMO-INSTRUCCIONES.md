# Datos Demo por Industria

El cargador demo ahora vive dentro de la aplicación.

## Uso

1. Entra a `index.html`.
2. Inicia sesión con tu cuenta de prueba.
3. Abre `Configuración`.
4. Baja a `Datos Demo`.
5. Elige el plan de prueba.
6. Pulsa la industria que quieres cargar.

Incluye demo para HVAC, Salón / Barbería, Transporte, Handyman, Limpieza y Construcción.

## Limpieza

El botón `Eliminar datos demo` borra solo registros marcados como demo. No borra registros reales.

## Nota

`demo-loader.html` se deja fuera del flujo principal. El demo oficial se carga desde Configuración.
